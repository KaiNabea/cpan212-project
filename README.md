# cpan212-project
An app for browsing and searching movies. You can add movies to your "Watched", "Favourites" and "Backlog" lists and leave reviews on how you felt about the movie.

Core Steps (Add as work continues)
1. Install relevant dependencies (Express, Express-validator, Mongoose, etc.)
2. Create module folder structure with routes, models, rules, middlewares, etc.
3. Find and fetch relevant movie database API and connect it to mongoDB
4. Check server works with Postman throughout the build process.

Other Steps
1. Create middleware to handle reviews
2. Create middleware to handle account setup and options
